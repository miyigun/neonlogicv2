# Neon Logic: Matrix Puzzle

**Neon Logic: Matrix Puzzle** is a browser-based **pattern-matching logic game** inspired by classic “Lights Out” mechanics.

You are shown a **TARGET** pattern (left). Your job is to reproduce it on the **INPUT/CONTROL** grid (right).  
Each click flips the selected cell and its **four neighbors** (up, down, left, right), making every move a small chain reaction.

---

## Features

### 1. Lights-Out Style Core Mechanic (5×5)
- Two 5×5 grids:
  - **TARGET:** the pattern you must match
  - **INPUT (CONTROL):** the grid you can interact with
- Clicking a cell toggles:
  - itself
  - up / down / left / right neighbors (within bounds)

### 2. Endless Level Progression (Procedural)
- Level starts at **1** and increases automatically after each win
- Each level generates a new solvable target pattern by applying random moves
- Difficulty scales with level (`difficulty = currentLevel + 2`)

### 3. Simple UX / Feedback
- “Level complete” notification banner
- Auto-advance to the next level after completion
- Manual controls:
  - **Reset:** clear player grid (target stays the same)
  - **Skip:** jump to the next level and generate a new target

### 4. Neon Retro UI
- Orbitron font + neon accent colors
- Clean, minimal single-page layout

## Getting Started

To run locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/miyigun/neonlogicv2.git
   ```
2. Enter the folder:
   ```bash
   cd neonlogicv2
   ```
3. Open `index.html` in your browser

> Recommended: run a local server for consistent behavior.
Example (Python):
```bash
python -m http.server 8000
```
Then open:
- `http://localhost:8000`

## 📌 Notes
- **Win condition:** the INPUT grid must match the TARGET grid cell-by-cell.
- **Reset** only clears the player grid; it does not regenerate the target.
- The target is generated by applying valid “moves,” which keeps levels solvable.
